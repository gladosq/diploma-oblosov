# Diploma-Oblosov

## Запуск
Перед запуском проекта у вас должны быть установлены [Node.js](https://nodejs.org/) и [npm](https://www.npmjs.com/).

Запуск dev-приложения:
```bash
# Установить зависимости
npm i
# Запустить локальный сервер
npm run dev
```

Сборка билда:
```bash
# Установить зависимости
npm i
# Собрать билд
npm run build
```
